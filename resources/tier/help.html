从夯到拉 · 使用指南

TierMaker 风格排行 · 一群一场 · 主题可选角色 / 武器 / 命座

作者 · NotBadSlime

夯 顶级 人上人 NPC 拉完了
1

开局

  • 仅在群聊使用；同一群同时只能有一场夯拉（不论类型），进行中时需先 #结束夯拉。
  • #角色从夯到拉 主题（与 #从夯到拉 主题 相同）→ 本场只排角色，不会出现武器或命座条目。
  • #武器从夯到拉 主题 → 本场只排武器。
  • #命座从夯到拉 主题 → 本场只排命座（每名角色 0～6 命各一条）；纯角色名视为 0 命。
  • 主题用于长图标题，建议简短（最多约 80 字）。
2

排位与飞榜

  • 指令形式：#…排夯 / 排顶级 等；移出所有档位可用 #…飞榜 / #…爬 / #…滚（同义）。写入内容仅取决于本场开局类型，不会跨类型误判。
  • 命座常用写法（可去空格）：钟离二命、纳西妲6命、胡桃C2、6命纳西妲、二命钟离、某某满命；中文数字 零〇一二两三四五六。
  • 批量：「排」前多项用空格或 ，、 分隔，例如命座场 #钟离 胡桃二命 排夯（钟离为 0 命、胡桃为 2 命）。
  • 每条只占一档；换档会自动从原档移除。
  • 主人专用（仅角色榜）：#其余角色排夯 或 #其他角色排顶级 等，将未排位池内全部实装角色一次性写入对应档位；非主人使用会提示无权限。
3

长图与未排位池

  • 每次成功排位后生成排行长图，标题旁标注 角色榜 / 武器榜 / 命座榜。
  • 命座图标：若喵喵素材中存在该角 icons/cons-1～cons-6，则优先显示命座图标；0 命或缺少素材时，用角色头像 + 右下角角标（数字或「满」）表示命数。
  • 底部未排位池仅本场未排位条目；过多时只展示前 72 项（命座场池子较大属正常）。
4

结束

  • #结束夯拉 或 #重置夯拉 结束当前这一场（任意类型）。
  • #结束角色夯拉 / #结束武器夯拉 / #结束命座夯拉 与上面等价，均为结束当前场。
